Differential Diagnosis Guide

Use the dropdowns below to view specific diagnostic logic and examples.

When a student scores low on Creative Problem Solving (SFCQ), distinguish these three profiles:

Profile A: Cognitive Constraint

Pattern: Low WM + Low Cognitive Flexibility + Normal/High Curiosity

Action: Scaffold externally (visual organizers). Do NOT insist they "think harder."

Profile B: Dispositional Barrier

Pattern: Normal WM + Low Cognitive Flexibility + Low Curiosity

Action: Build tolerance for ambiguity. Check motivation.

Profile C: Exploratory Deficit

Pattern: Normal WM + Normal Flexibility + Low Curiosity + Low SFCQ-Knowledge

Action: Ignite curiosity through exposure to diverse perspectives.

Administration & Scoring

Expand the sections below for detailed protocols.

T3EndEF + Core Skills + SDT + IRI + Context25–35 min

Each sitting is divided into sprints of 10–15 minutes to minimize cognitive fatigue.

T1: Intake Sprints

  • T1a: EF + SDT baseline (48 items)
  • T1b: Core Skills (47 items)
  • T1c: Interpersonal + Cultural (36 items)

T2: Check-in Sprints

  • T2a: Core Skills I (35 items)
  • T2b: Core Skills II (24 items)

T3: Outcomes Sprints

  • T3a: EF + SDT (48 items)
  • T3b: Core Skills (47 items)
  • T3c: Contextual + Interpersonal (39 items)
  • Score carry-forward: T2 reports display the most recent EF and SDT scores from T1 alongside current skill scores.
  • Timestamps per lens: Each score carries its collection date (e.g., "EF scores from T1 — 6 weeks ago").
  • Three-point trajectory: The engine supports slope/trend calculation across T1/T2/T3.
  • Partial battery handling: The system treats missing instruments at T2 as by-design, not as missing data.
